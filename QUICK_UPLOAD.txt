ПОШАГОВО (ЕСЛИ ПЕРВЫЙ РАЗ С GIT):

1. Откройте Командную строку (cmd.exe)

2. Выполните эти команды по порядку:

   git config --global user.name "Ваше имя"
   git config --global user.email "ваша.почта@gmail.com"

3. Клонируйте репозиторий:

   cd C:\Users
   git clone https://github.com/aleksey341/Parser_Autonomera.git
   cd Parser_Autonomera

4. Откройте папку C:\Users\Parser_Autonomera в проводнике

5. Скопируйте туда все файлы из C:\Users\cobra\
   (кроме node_modules и .git)

6. Вернитесь в cmd и выполните:

   git add .
   git commit -m "Add complete parser"
   git push origin main

5. Используйте его как пароль при git push
