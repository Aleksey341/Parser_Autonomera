После того, как вы поделитесь ссылкой, коллеги смогут:

📌 СПОСОБ 1: Использовать онлайн
   1. Откроют ссылку в браузере (например: https://autonomera777-parser.up.railway.app/)
   2. Сразу начнут парсить, без установки чего-либо
   3. Смогут скачивать CSV файлы с результатами

📌 СПОСОБ 2: Установить локально
   1. Клонируют репозиторий:
      git clone https://github.com/YOUR_USERNAME/autonomera777-parser.git
   2. Установят зависимости:
      npm install
   3. Запустят:
      npm start
   4. Откроют http://localhost:3000

📌 СПОСОБ 3: Развернуть на своем сервере
   1. Клонируют репозиторий
   2. Следуют инструкциям в файле DEPLOYMENT.md
   3. Развертывают на своем VPS/хостинге
